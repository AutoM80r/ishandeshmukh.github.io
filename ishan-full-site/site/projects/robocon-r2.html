Robocon R2 Autonomous

Fully autonomous robot for DD Robocon 2025 with YOLOv5 silo detection, ROS RTAB-Map SLAM, and 3D-printed 3-finger gripper.

Jetson NanoROS2YOLOv5SLAMRobocon 2025
overview

R2 is the autonomous companion robot to R1. It navigates the arena independently, detects silo targets using a custom-trained YOLOv5 model, and plants seedlings using a 3D-printed 3-finger gripper — all without any human input during the run.

PERCEPTION
Zed 2i depth camera feeds point cloud to RTAB-Map. TCS230 identifies seedling colors. LSS05 line sensors for arena boundary.
DETECTION
Custom YOLOv5 model trained on silo images from arena. Runs on Jetson Nano GPU. ~40ms inference.
LOCALISATION
ROS RTAB-Map SLAM with visual-inertial odometry. Nav stack handles path planning and obstacle avoidance.
MANIPULATION
3D-printed 3-finger underactuated gripper. Servo-driven with position feedback. Handles variable seedling sizes.
COMMS
ROS2 topics between Jetson Nano (high-level) and STM32 (low-level motor control) over UART bridge.
